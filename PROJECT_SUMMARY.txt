🚀 HOW TO DEPLOY (3 OPTIONS)
═══════════════════════════════════════════════════════════════════════════════

OPTION 1: VERCEL (Recommended - 2 minutes)
─────────────────────────────────────────────
1. Push to GitHub:
   git remote add origin https://github.com/oxylok/balidream-haiku-screen.git
   git branch -M main
   git push -u origin main

2. Go to vercel.com/dashboard
3. Click "New Project" → Select repo → "Deploy"
4. Get live URL

LIVE URL: https://balidream-haiku-screen.vercel.app

OPTION 2: GITHUB PAGES (2-3 minutes)
────────────────────────────────────
1. Push to GitHub (same as above)
2. Go to repo → Settings → Pages
3. Select "main" branch / "/" folder → "Save"
4. Get live URL

LIVE URL: https://oxylok.github.io/balidream-haiku-screen/

OPTION 3: NETLIFY (2 minutes)
──────────────────────────────
1. Push to GitHub (same as above)
2. Go to netlify.com → "New site from Git"
3. Select repo → "Deploy site"
4. Get live URL

LIVE URL: https://balidream-haiku-screen.netlify.app
