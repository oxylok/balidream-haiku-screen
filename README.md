# Bali Dream — Luxury Travel Landing Page

A premium single-page website for Bali Dream travel experiences, built with HTML5 + Tailwind CSS.

## Features

- **Responsive Design**: Optimized for 375px (mobile), 768px (tablet), and 1920px (desktop) viewports
- **Accessible**: WCAG AA contrast compliance, semantic HTML, alt-text ready
- **Performance**: Lighthouse-optimized, fast-loading, no external dependencies except Tailwind
- **SEO Ready**: Proper title, meta description, h1, semantic markup

## Sections

- Hero section with compelling CTA
- Featured experience highlight
- Three-card experience selector
- Rice terrace spotlight section
- Four-image gallery grid
- Booking/reservation interface
- Three customer testimonials with ratings
- FAQ accordion
- Call-to-action section
- Footer with links and social

## Technology Stack

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first styling via CDN
- **Google Fonts** - Cormorant Garamond (serif) + Poppins (sans)
- **Vanilla JavaScript** - Smooth scrolling, nav effects

## Color Palette

- Primary: #DC6B5D (Rust/Coral)
- Secondary: #2B5A6E (Teal)
- Accent: #F4C430 (Gold)
- Background: #F8F7F6
